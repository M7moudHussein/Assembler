To run the Assembler, You must use the CMD
1)Open the CMD from the same directory
2)Type "Assembler.exe input.txt"

You can replace 'input.txt' by the name of the input file you are using.
The output is generated in a folder named output in the directory of the assembler.
But you must first create the folder or the directory 'output'.
It'll include both files, The Listing and the Object code.
